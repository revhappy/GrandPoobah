Welcome to our CAD software! This platform allows you to create and edit 3D models and 2D drawings.

3D Models

To create a new 3D model, simply type in the prompt "create 3D model" followed by the name of the object you want to create (e.g. "create 3D model ball").

3D Model Canvas

Here you can see your 3D model being generated in real-time. Use the navigation menu on the right to move, rotate, and scale the object.
